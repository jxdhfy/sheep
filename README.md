# sheep
C语言的知识点

1.必备知识

  1.1 基本数据类型（基本类型、构造类型、指针类型、空类型）
  
      基本类型：int、char、float、double
      构造类型：[]、struct、union、enum
      指针类型：*
      空类型：void
      
  1.2 标识符是由字母、数字、下划线组成的，且不以数字开头。区分大小写，含义不同。

  1.3 各个表示的含义
  
      %c——打印一个字符格式的数据
      %s——打印字符串
      %p——打印以地址的形式
      %d——打印整型十进制的数据
      %u——打印无符号十进制整数
      %f——打印浮点型的数据
