C语言的知识点

1.必备知识

  1.1 基本数据类型（基本类型、构造类型、指针类型、空类型）
  
      基本类型：int、char、float、double
      构造类型：[]、struct、union、enum
      指针类型：*
      空类型：void
      
  1.2 标识符是由字母、数字、下划线组成的，且不以数字开头。区分大小写，含义不同。

  1.3 各个表示的含义
  
      %c——打印一个字符格式的数据
      %s——打印字符串
      %p——打印以地址的形式
      %d——打印整型十进制的数据
      %u——打印无符号十进制整数
      %f——打印浮点型的数据

  1.4 转义字符

      \n：换行
      \b：退格
      \\：打印出\
      \'：打印出'
      \"：打印出"

2.运算符

  2.1 算术运算符（左结合）
  
      一共有五种，分别是加法 +、减法 -、乘法 *、除法 /、求余 %
      
      除法/ ：其中一个操作数必须加上 . ；若不能整除，结果必为整数舍去小数。例：5./2 = 2 
      求余% ：两个操作数都为整型，且运算结果的符号与被除数的符号相同。例：5%2 = 1，-5%3= -2

  2.2 移位运算符（左结合）

      移位移的是二进制整型，一共有两种，分别是左移 <<、右移 >>
      
      int a = 1;               //a=01
      int b =a<<1;             //b=10
      printf("b=%d\n",b);      //b=2

  2.3 位逻辑运算符（左结合）

      位逻辑运算符也是用二进制计算，一共有三种，分别是按位与 &、按位或 |、按位异或 ^
      按位与 &  ————只有当两个操作数都为真（1），结果才为真（1）
      按位或 |  ————只需要其中一个操作数为真（1），结果就为真（1）
      按位异或 ^  ————先将两个操作数用二进制表示，各数上下对齐。对应的二进制位相同，则为假（0）；对应的二进制位不同，则为真（1）

      int a = 3;                            //a=011
      int b = 5;                            //b=101
      int c = a & b;                        //c=011 & 101 = 001
      int d = a | b;                        //d=011 | 101 = 111
      int e = a ^ b;                        //e=011 ^ 101 = 110
      printf("c=%d,d=%d,e=%d\n"c,d,e);      //c=1,d=7,e=6

  2.4 赋值运算符（左结合）

      ==（判断是否相等）、!=（不相等）

  2.5 复合赋值符（右结合）

      +=、-=、*=、/=、%=、>>=、<<=、&=、^=、|=

  2.6 单目操作符

      !（逻辑非）、+、-、&、sizeof（计算变量或类型所占的空间大小，单位为字节）、~（按位取反）、++、--、*（指针类型）
      int a =10;
      printf("%d\n",sizeof(a));      //结果都为4
      printf("%d\n",sizeof(int));    
      printf("%d\n",sizeof a);
